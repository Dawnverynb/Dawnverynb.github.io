变量

  1. 变量存在的意义：方便我们管理内存空间
  2. 变量创建的语法：数据类型 变量名 = 变量初始值

常量

  1. 宏常量： #define 常量名 常量值
  2. const 修饰的变量:const 数据类型 常量名 = 常量值

关键字

  1. 作用：关键字是c++预先保留的单词（标识符）
  2. 在定义变量和常量的时候，不要用关键字

    int int = 10//会报错 //第二个int是关键字不能作为变量名称
  3. 标识符的命名规则
    标识符不能是*关键字
    标识符只能由字母数字和下划线构成
    标识符第一个字符只能是**字母和下划线
    标识符*区分大小写

数据类型

数据类型存在的意义：给变量分配合理的空间

整形

作用：整形变量表示的是整数类型的数据

数据类型 占用空间 取值范围
short(短整型) 2字节 (-2^15 ~ 2^15-1)（-32768~32767）
int(整型) 4字节 (-2^31 ~2^31-1)
long(长整形) Windows为4字节，Linux为4字节(32位)，8字节(64位) (-2^31 ~2^31-1)
long long(长长整形) 8字节 (-2^63 ~2^63-1)

sizeof关键字

作用：利用sizeof关键字可以统计数据类型所占内存大小
short < int<= long <= long long

实型（浮点型）

作用：表示小数

  1. 单精度: float 七位有效数字 4字节
  2. 双精度：double 15~16位有效数字 8字节
    //默认情况下系统会认为小数是双精度，所以要在小数后加f
    //默认情况下，输出一个小数会显示6位有效数字

字符型

作用:字符型变量用于显示单个字符
语法： char ch = 'a'；

注意1：在显示字符变量时，要用单引号括起来，不要用双引号
注意2：单引号内只能有一个字符，不能有多个

cout <<(int)ch << endl； //查看字符a对应的ASCLL码
ch = 97；//可以直接用ASCLL给字符型变量赋值
65是A

转义字符

作用：用于表示一些不能显示出来的ASCLL字符
现阶段常用的有：\n \\ \t

  • \n :换行符
  • \:反斜杠
  • \t ：水平制表符占八个位置

字符串型

作用：用于表示一串字符

  1. c风格字符串： char 变量名 [] = "字符串值"

    c风格的字符串要用[]

  2. c++风格的字符串：string 变量名 = "字符串值"

c++风格的字符串要在头文件加#include

布尔数据类型

作用：布尔数据类型表示真假
boll类型有两个值：

  • true –真 // 本质是1
  • false –假 // 本质是0
    bool类型占一个字节大小

    除了0（false）之外的所有数值都表示为1（true）

数据的输入

作用：用于从键盘获取数据
关键字： cin
语法： cin>> 变量

运算符

作用：用于执行代码的运算

算术运算符

作用：用于处理四则运算
